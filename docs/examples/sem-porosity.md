# SEM Porosity Example

## Real SEM Image: ZnSe Porous Surface

Analysis of a real scanning electron microscopy image of a porous ZnSe surface.

```bash
poroscope analyze examples/data/sem_samples/sem_znse_porous.jpg \
  --pixel-size 0.5 --unit um --pores dark
```

**Result**: 68.96% porosity, 413 pores detected.
